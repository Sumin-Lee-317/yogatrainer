# yogatrainer

## General info
This code to develop a application for yoda trainer.
The trainee will follow the target pose.
On the screen, we will display: the score, the comment, and the suggestion to have a perfect pose as target
	
## Technologies
+ Mediapipe
+ Cosine Similarity
